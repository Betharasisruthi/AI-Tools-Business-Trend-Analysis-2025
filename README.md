# AI Tools for Business in India – 2025 Trend Analysis

## Overview

This project analyzes Google Trends data for "AI tools for business" in India during 2025.

## Objectives

* Analyze search interest trends
* Identify peak search periods
* Generate business insights
* Provide marketing recommendations

## Tools Used

* Google Trends
* Google Sheets
* Google Docs

## Key Findings

* Overall trend: Upward
* Peak month: November 2025
* Peak search score: 100
* Strongest growth period: November–December 2025

## Business Recommendations

* Launch educational email campaigns
* Promote AI productivity use cases
* Offer product demonstrations
* Share customer success stories
